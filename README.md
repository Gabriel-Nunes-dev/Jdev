# Jdev
Sistema de Gerenciamento Escolar em Java

O projeto de Sistema de Gerenciamento Escolar em Java é uma aplicação de console que permite aos usuários realizar o cadastro de alunos e calcular suas médias de notas. Os usuários podem adicionar informações dos alunos, como nome, matrícula e notas em disciplinas específicas. A aplicação deve calcular automaticamente a média das notas e fornecer feedback sobre o desempenho do aluno. Além disso, ela pode incluir recursos para exibir a lista de alunos cadastrados, permitir a edição de informações e a remoção de alunos. O projeto tem como objetivo aprofundar seus conhecimentos em Java, classes, objetos, estruturas de controle e cálculos simples, enquanto simula um sistema de gerenciamento escolar básico.
